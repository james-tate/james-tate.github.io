Ghidra -- A quick start guide

The NSA recently released Ghidra, a powerful open source software reverse engineering tool. Since the release I have played around with Ghidra for a number of hours and found it to be pretty comparable to existing tools that are used for software reverse engineering. I have also read a number of blog post and articles that try to compare Ghidra to these other tools, expecially Ida Pro. I didn't want to try to do a comparison since every tool in the RE space has it's own advantages and disadvantages. If doing a straight comparison between tools, Ghidra does basiclly everything other tools do, however it displays information differently to the user. Knowing how to use and interpret this displayed information to come to a conclusion is the advantage of one tool over the next. This really comes from experiance, and I think that's the advantage the other tools have over Ghidra at this time. Instead of trying to make a comparison of the tools, I wanted this post to be about guiding someone into getting Ghidra setup with a quick example.
